Apa itu Software?

Software atau perangkat lunak adalah kumpulan program, data, dan dokumentasi yang menginstruksikan komputer atau perangkat elektronik tentang cara melakukan tugas tertentu. Ini adalah komponen sistem komputer yang tidak berwujud, berbeda dengan perangkat keras, yang mengacu pada komponen fisik. Perangkat lunak dapat dikategorikan menjadi beberapa jenis, masing-masing memiliki tujuan dan fungsi yang berbeda. Software terdiri dari beberapa komponen yaitu Program Ini adalah kumpulan instruksi yang dijalankan komputer untuk melakukan tugas. Program dapat berkisar dari skrip sederhana hingga aplikasi kompleks yang mengelola berbagai fungsi dalam suatu sistem. Data Ini mencakup informasi yang dimanipulasi oleh program untuk menghasilkan hasil yang berarti. Data bisa dalam berbagai bentuk, seperti teks, gambar, audio, atau video, dan penting untuk pengoperasian aplikasi perangkat lunak. Dokumentasi Ini mencakup manual, panduan pengguna, dan materi tertulis lainnya yang menjelaskan cara menggunakan perangkat lunak, fitur-fiturnya, dan fungsinya. Dokumentasi sangat penting bagi pengguna untuk memahami dan memanfaatkan perangkat lunak secara efektif. Perangkat lunak memainkan peran penting dalam masyarakat modern, memungkinkan berbagai fungsi di berbagai sektor, termasuk bisnis, pendidikan, perawatan kesehatan, dan hiburan. Ini meningkatkan produktivitas, memfasilitasi komunikasi, dan menyediakan alat untuk analisis data dan pengambilan keputusan. Seiring dengan terus berkembangnya teknologi, permintaan akan solusi perangkat lunak inovatif pun semakin meningkat, menjadikan rekayasa perangkat lunak sebagai bidang yang vital.

System Software

System software atau perangkat lunak sistem adalah perangkat lunak yang berfungsi sebagai perantara antara perangkat keras (hardware) dan perangkat lunak aplikasi (application software). Perangkat lunak ini sangat penting dalam sistem tertanam (embedded systems), karena memungkinkan operasi yang efisien, pengelolaan sumber daya, serta integrasi dengan perangkat keras. Komponen utama System Software adalam Sistem Tertanam antara lain adalah Sistem Operasi Tertanam (Embedded Operating System) yang berfungsi menangani proses I/O, penjadwalan dan context switching, memiliki kemampuan real-time dalam mengatasi tugas dalam waktu tertentu, serta mendukung akses eksklusif ke sumber daya dengan protokol seperti priority inheritance, priority ceilling, dan stack resource protocols. Ada syarat yang harus dipenuhi untuk Sistem Operasi Tertanam yaitu ukuran yang kecil dan bisa dikonfigurasi, efisiensi dalam pengelolaan sumber daya, dan keamanan serta ketahanan terhadap serangan. Contoh implementasi System Software dalam System Tertanam yaitu Embedded Linux seperti pada router dan TV pintar dan Real-Time Operating System - RTOS pada ERIKA dan FreeRTOS Adapun Middleware sebagai komponen utama System Software yang berada diantara sistem operasi dan aplikasi lalu berfungsi sebagai komunikasi antara berbagai macam komponen dalam sistem tertanam. Lalu terdapat Hardware Abstraction Layer (HAL) yang memisahkan aplikasi dari perangkat keras yang mendasarinya serta memungkinkan fleksibilitas dalam pengembangan dan portabilitas peranngkat lunak ke berbagai perangkat keras.

Application Software

Application Software atau perangkat lunak aplikasi dirancang untuk membantu pengguna melakukan tugas atau fungsi tertentu, seperti pengolah kata, pengelolaan data, atau desain grafis. Berbeda dengan perangkat lunak sistem, yang mengelola perangkat keras dan sumber daya sistem, perangkat lunak aplikasi secara langsung memenuhi kebutuhan pengguna dan meningkatkan produktivitas. Untuk mengimplementasikan perangkat lunak aplikasi, pengembang mengikuti proses terstruktur yang mencakup pengumpulan persyaratan, desain, pengkodean, pengujian, dan penerapan. Hal ini sering kali melibatkan penggunaan metode rekayasa perangkat lunak untuk memastikan kualitas, efisiensi, dan kepuasan pengguna sepanjang siklus pengembangan. Contoh umum perangkat lunak aplikasi adalah Microsoft Word, yang memungkinkan pengguna membuat, mengedit, dan memformat dokumen teks. Ini mencakup fitur-fitur seperti pemeriksaan ejaan, templat, dan alat kolaborasi, menjadikannya alat serbaguna untuk penggunaan pribadi dan profesional.

Hubungan Antara System Software, dan Application Software

Berikut ini adalah penjelasan mengenai hubungan antara System Software dan Application Software :

  • Ketergantungan: Application Software bergantung pada System Software untuk berfungsi. Tanpa Operating System (sejenis System Software), Application Software tidak dapat berjalan, karena memerlukan sistem yang mendasarinya untuk mengelola sumber daya perangkat keras.
  • Interaksi: System Software menyediakan lingkungan dan layanan yang diperlukan Application Software untuk beroperasi. Misalnya, ketika pengguna membuka pengolah kata, sistem operasi mengalokasikan memori dan mengelola operasi I/O untuk memfasilitasi fungsionalitas aplikasi.
  • Tingkatan: System Software berada pada tingkat dasar, menyediakan layanan dan sumber daya penting yang dibangun oleh Application Software untuk memberikan fungsionalitas tertentu kepada pengguna.
